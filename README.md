# 📖 BookForge

La piattaforma AI per scrittori italiani — scrivi, illustra e pubblica su Amazon KDP.

## Stack
- React 18 + Vite
- Claude Sonnet API (Anthropic)
- Vercel hosting

## Deploy
1. Fork/clone questo repo
2. Collega a Vercel
3. Deploy automatico ad ogni push

## Sviluppo locale
```bash
npm install
npm run dev
```
